# LinkedIn Post Tags Reference

When creating LinkedIn posts in `src/content/linkedin/`, use these common tags:

## Recommended Tags

- `ProductManagement` - For posts about product strategy, roadmaps, feature development
- `AI` - For posts about artificial intelligence, machine learning, AI products
- `Startup` - For posts about startup life, entrepreneurship, founding
- `Leadership` - For posts about team management, leadership principles
- `SystemsThinking` - For posts about systems design, architecture, scalable solutions

## Usage Example

```mdx
---
title: "Building AI-Powered Products"
date: 2024-01-15
tags: ["AI", "ProductManagement", "SystemsThinking"]
canonicalUrl: "https://linkedin.com/posts/..."
---

Your post content here...
```

## Tag Guidelines

- Use PascalCase (e.g., `ProductManagement` not `product-management`)
- Be consistent with tag names
- Use 2-4 tags per post for best organization
- Tags will appear as badges on the post pages
